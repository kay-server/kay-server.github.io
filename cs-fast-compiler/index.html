    C# Fast-Compiler - подпольная IDE для C#

    Author: DosX Kay Software hosting

    О программе

    Бесплатная IDE для Windows со встроенным компилятором C#. Рассчитана для быстрого создания консольных проектов или тестирования исходных кодов.

    Скачать

    Вы можете получить и установить C# Fast-Compiler на свой компьютер, скачав необходимые файлы здесь.

    Программа: Скачать портативную версию (AnyCPU)

    Исходный код проекта: Скачать исходный код (C#)

    Компиляция

    Если код написан правильно, то в папке bin появится результат компиляции и сразу запустится. Имя файла зависит от написанного названия проекта в программе. Если при построении возникла ошибка, то выведется сообщение с информацией об исключении
