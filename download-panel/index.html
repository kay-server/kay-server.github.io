    Файлообменник без ограничений и рекламы

    Author: DosX Главная страница

    Kay Download Panel

    Бесплатная простая панель для публикации и скачивания файлов. Имеет минималистичный стиль. Полностью отсутствует реклама.

    Как я могу загрузить файл?

    Но как же сгенерировать такую ссылку? На деле всё куда проще, чем кажется. Эта страница использует GitHub в качестве хостинга файлов. Вам требуется лишь создать открытый репозиторий с названием "kay-software-cloud" (Как создать репозиторий на GitHub) и добавить туда все необходимые файлы через сайт или Desktop-версию GitHub. Проще говоря - сайт ищет указанный в ссылке файл в указанном репозитории, куда Вы его и загружаете.

    Для автоматического создания ссылки на файл воспользуйтесь встроенным генератором на главной странице сервиса ("File link generator"). Вам требуется лишь ввести необходимые данные (ник на GitHub) и название загруженного файла, после чего скопировать полученный результат.

    Создание ссылки вручную

    Достаточно лишь заполнить необходимые данные внутри для создания подобной ссылки: "https://kay-software.ru/download/?file=ИМЯ_ФАЙЛА&from=ИМЯ_АККАУНТА".

    После всех этих простых манипуляций Вы можете делиться ссылкой со своими друзьями.

    Важно!

    Рекомендуется загружать архивы/бинарники, поскольку текстовые файлы многие браузеры вместо скачивания сразу открывают. В случае ошибок "404", "400" и т.д. пользователя перенаправит на страницу GitHub с текстом ошибки. (после нажатия на кнопку). Если аккаунт не указан, то файл будет автоматически скачиваться с нашего сервера.

    Если Вы видите галочку у аккаунта, с которого был загружен файл, то знайте - это проверенное лицо. Все файлы, загружаемые от его имени проверены и не содержат вредоносный код. Верификацию аккаунта можно получить только если Вы: известная личность, представитель юридически-зарегистрированной компании, сотрудник Kay Software или спонсор.
